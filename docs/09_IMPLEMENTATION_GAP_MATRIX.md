## 다음 액션 — 우선순위 순

### 🔴 HIGH (즉시 개발 필요)

1. **관리자 페이지 RBAC 미들웨어 적용**
   - `/admin/**` 경로에 ADMIN 역할만 접근 허용 (middleware.ts 또는 layout.tsx 레벨)
   - `/settings` 프로필 수정 기능 구현 (이름, 비밀번호 변경)

2. **설정 페이지 기능 구현**
   - 프로필 수정 (이름, 비밀번호 변경)
   - 알림 수신 설정 (이메일 ON/OFF)

3. **대시보드 차트 시각화**
   - 단계별 프로젝트 분포 → 가로 막대 차트
   - 월별 완료 실적 → 라인/바 차트
   - (`recharts` 또는 `chart.js` 활용)

### 🟠 MEDIUM (다음 스프린트)

4. **프로젝트 신규 생성 폼 — 유사 프로젝트 추천 연동**
   - 사업장+공정+품목 선택 시 실시간 유사 프로젝트 추천 카드 표시
   - `/api/v1/projects/similar` 연동

5. **문서 업로드 UX 개선**
   - 드래그&드롭 영역 추가
   - 업로드 진행 표시(progress bar)
   - 이미지 파일 썸네일 미리보기

6. **검색 페이지 UX 개선**
   - Enter 키 검색 지원
   - 검색 결과 없을 때 빈 상태 안내
   - 날짜 범위 필터 추가

7. **회원가입/비밀번호 재설정 운영 정책 완성**
   - 이메일 인증 플로우 (Resend 또는 Nodemailer)
   - 재설정 토큰 만료 처리

### 🟡 LOW (안정화 단계)

8. **이메일 알림 연동**
   - 단계 전환 시 담당자 이메일 발송
   - Resend API 연동 (`@/lib/email.ts` 구현)

9. **Google Drive 마이그레이션 실연동**
   - OAuth 토큰 처리
   - 파일 분류→Supabase Storage 이전 스크립트 완성

10. **shadcn/ui 컴포넌트 일관성 적용**
    - 현재 raw HTML(`<input>`, `<select>`, `<button>`) → shadcn/ui `Input`, `Select`, `Button`으로 전환
    - 삭제 액션에 `AlertDialog` 확인 모달 추가

11. **반응형(모바일) UI 보완**
    - 프로젝트 목록 테이블 → 모바일에서 카드 레이아웃
    - 대시보드 그리드 breakpoint 조정
